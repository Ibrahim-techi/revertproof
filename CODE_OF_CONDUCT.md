# Code of Conduct

RevertProof follows the Contributor Covenant Code of Conduct.

## Our Standards

Project spaces should be professional, welcoming, and focused on useful technical work. Examples of expected behavior include:

- using respectful language;
- accepting constructive feedback;
- staying focused on the project mission;
- helping maintainers keep discussions actionable.

Unacceptable behavior includes harassment, personal attacks, spam, or knowingly misleading claims.

## Enforcement

Maintainers may remove comments, close issues, or restrict participation when behavior harms the project or its contributors.

Report conduct concerns privately to the project maintainer.
